# 应用及其实例

每个 Vue 应用都是通过 createApp 函数创建一个新的应用实例开始的。Vue 官方习惯称应用为 app。

# 挂载（mount）应用

挂载应用就是把应用挂到一个元素上，并在此元素上进行实例化。

# 示例
const app = Vue.createApp({ /* 选项 */ })
const vm = app.mount('#app')

# 全局注册

组件全局注册之后可以用在应用内任何组件实例的模板中。

注册方式为，用 app.component 创建组件：

app.component('my-component-name', { /* 选项 */ })
app.mount('#app')

# 局部注册

局部注册是让注册的组件只能在当前“作用域”内使用，组件的父子组件要使用必须再次导入，这样子可以一定程度地辅助构建系统减少最终的构建结果的文件大小。

可以先在一个文件声明组件，然后再在别的组件局部注册这个组件。

const ComponentA = {
  /* ... */
}

const ComponentB = {
  components: {
    'component-a': ComponentA
  }
  // ...
}

# 在模块系统中局部注册

方法与局部注册类似，但需要先导入组件。建议创建个 components 目录用于放置组件，然后在注册前导入所有要用的组件：

import ComponentA from './ComponentA'
import ComponentC from './ComponentC'

export default {
  components: { // 类似局部注册
    ComponentA,
    ComponentC
  }
}

# Prop

Prop 是可以在组件上注册的一些自定义 attribute，类似于函数的参数。这些 prop 可以在使用组件时附加。

如下面例子的 title：

app.component('blog-post', {
    props: ['title'],
    template: `<h4>{{ title }}</h4>`
})

app.mount('#blog-post-demo')
