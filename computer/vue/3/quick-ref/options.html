# 选项 API | Vue 3 快速参考

# Data

控制数据的选项。

# data

  • 类型： function(vm): Object

  • 细节:

    data 用于定义组件的数据。该函数返回组件实例的 data 对象。实例创建时会被调用，然后对象所有顶级 property 也会直接通过组件实例暴露出来：

    const vm = createApp({
    data() { return { a: 1 } }
}).mount('#app')

console.log(vm.a) // => 1

    一旦 property 被侦听后，根数据对象就不能再被添加响应式的 property。所以应在创建实例前先声明所有的根级响应式 property。

    警告

    以 _ 或 $ 开头的 property 不会被组件实例代理，因为它们可能会和 Vue 内置的 property 或方法冲突。可以使用如 vm.$data._property 的方式访问这些 property。

    如果为 data property 使用了箭头函数，则可以通过该函数的第一个参数来访问实例：

    data: vm => ({ a: vm.myProp })

# 访问原始数据对象

实例创建之后，组件实例代理了 data 上对象上所有的 property。因此可以通过 vm 或 vm.$data 访问原始数据对象。

# methods

  • 类型： { [key: string]: Function }

  • 细节:

    methods 用于定义组件的方法。其所有方法都将被混入到组件实例中，因此可以直接通过组件实例（vm）访问或在指令表达式中使用。方法中的 this 自动绑定为组件实例。如：

    const vm = createApp({
    data() { return { a: 1 } },
    methods: {
        plus() { this.a++ }
    }
}).mount('#app')

vm.plus()
console.log(vm.a) // => 2

# computed

  • 类型： 设 T 为该计算属性的类型，

    { 
    [key: string]: (vm) => T // 单 getter 的形式
    | { // 设置 getter 和 setter 的形式
        get: (vm) => T,
        set: (v: T, vm) => undefined
    }
}

  • 细节：

    computed 选项用于定义计算属性。计算属性将会以属性的形式被混入到组件实例中。所有 getter 和 setter 的 this 上下文自动地绑定为组件实例。

    如果为计算属性使用了箭头函数，则可以通过该函数的第一个参数来访问实例。

  • 示例：

    const vm = createApp({
data() { return { a: 1 } },
computed: {
    aDouble() { return this.a * 2 },
    aPlus: {
    get() { return this.a + 1 },
    set(v) { this.a = v - 1 }
    }
}
}).mount('#app')

console.log(vm.aPlus) // => 2
vm.aPlus = 3 // a = 3 - 1 => 2
console.log(vm.a) // => 2
console.log(vm.aDouble) // => 4
