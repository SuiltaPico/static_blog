# VuePress

# 介绍

这是一个用于搭建博客的引擎（比如这个博客就是 VuePress 搭建起来的）。能自定义插件和主题（你得懂点 VuePress 原理），也可以用别人写好的（新手推荐）。

# 为什么推荐

虽然我没有用过别的引擎，但这个引擎给我的感觉就是，特别轻便快捷。

轻便在于，其安装完成后没有出现一大堆的各种配置文件，没有烦人的视觉干扰，并且即装即用，你只需要在 docs/ 文件夹里面写 .md 文件就好了。往后所有的配置只需要操作一个 VuePress.config.ts，且其 官方文档在新窗口打开 写的也十分简洁明确，十分方便。

快捷是因为其使用了模块热重载在新窗口打开 贼快的 Vite在新窗口打开，做到了就像 Markdown 编辑器一样的内容更新实时渲染。但是因为能装插件并且跑得也挺快，又比 Markdown 编辑器更胜一筹。

不过总体来说这里还是充满着主观臆断，之后我会整个活，好好对比一下各个博客引擎的。

# 注意

不建议去用 VuePress v1，其构建在 Webpack 之上，模块热重载一次可能要好几秒！尽量使用 VuePress v2 或以上的版本。

# 和 VitePress 的区别

官方文档 differences-from-vuepress在新窗口打开

VitePress 旨在拥有创作文档所需的最低限度的功能，其他功能被移到主题。

VuePress 具有更多开箱即用的功能或由其插件生态系统启用。

大致意思可能是，VuePress 给你提供一个不错的地基和一堆小积木元件，你可以自己喜欢怎么搭就怎么搭，甚至还可以根据规范自己做积木元件；而 VitePress 就像直接选一个别人做好的“产品”（主题），里面的东西你改不了多少，“产品”质量完全取决于开发者。
